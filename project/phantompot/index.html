PhantomPot

Jul 2025 - Oct 2025

Splunk Python Docker

Overview

PhantomPot is an advanced honeypot system designed to simulate vulnerable services and capture attacker behavior for threat analysis and security research purposes.

Key Features

  • Simulated vulnerable services including HTTP, SSH, and FTP protocols
  • Real-time capture of attacker credentials and malicious payloads
  • GeoIP tracking to identify attacker locations and patterns
  • Automated email alerts for immediate threat notification
  • Integration with Splunk for comprehensive log analysis and visualization

Technical Implementation

The system is built using Python for the core honeypot services, containerized with Docker for easy deployment and isolation. All captured data is forwarded to Splunk for real-time monitoring and historical analysis.

Impact

PhantomPot provides valuable insights into attacker methodologies, helping security teams understand threat vectors and improve their defensive strategies.
